// バックエンドサーバー（Cloudflare D1 の API）を起動
// --ip 0.0.0.0 でLAN内のスマホ・タブレットからも接続できるようになります

cd customer-api-prototype
npm run dev



本番環境（URL）: npx wrangler deploy が完了した瞬間から、本番のWebサイトは、先ほどインポートしたD1データベースを参照するようになります。
ローカル環境: 自分のPCで npx wrangler dev を実行したときは、これまで通りPC内のローカルDBを見に行きます。
※もしローカル開発時も本番のDBを見たい場合は # 一度 Ctrl+C で停止してから、以下で再起動
npx wrangler dev --remote --ip 0.0.0.0
 と打ちます。

customer-api-prototype ディレクトリはGitリポジトリになっていないため、以下のコマンドを実行してバックエンドをCloudflare Workersへデプロイ

cd customer-api-prototype
npm run deploy


Cloudflare APIへの接続で認証エラー（Authentication error [code: 10000]）が発生した場合、

一度ログアウトしてから再ログインすると正常に動作する場合がある

npx wrangler logout

npx wrangler login

npm run deploy
